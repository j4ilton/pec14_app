INSTRUÇÕES PARA CONFIGURAR O ÍCONE
===================================

1. Salve a imagem do ícone (PEC 14) nesta pasta com o nome: icon.png
2. A imagem deve ter pelo menos 1024x1024 pixels para melhor qualidade
3. Após salvar a imagem, execute o comando abaixo no terminal:

   flutter pub get
   dart run flutter_launcher_icons

4. Isso gerará automaticamente os ícones para todas as plataformas (Android, iOS, Web, etc.)

IMPORTANTE: Não se esqueça de salvar a imagem como "icon.png" nesta pasta antes de executar os comandos.
